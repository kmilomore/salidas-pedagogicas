# Componente: Stepper

## Archivo
- `src/components/nueva-salida/Stepper.tsx`

## Rol
Representar la navegación visual del wizard y el estado de avance del usuario.

## Responsabilidades
- Marcar paso actual, pasos previos y disponibilidad de navegación.
- Mostrar `helperText` y label contextual del botón siguiente.
- Deshabilitar navegación durante estados ocupados como cálculo de ruta o guardado.
